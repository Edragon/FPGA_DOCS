Sample behavioral waveforms for design file dct_pll.v

The following waveforms show the behavior of altpll megafunction for the chosen set of parameters in design dct_pll.v. The design dct_pll.v has Stratix II AUTO pll configured in NORMAL mode The primary clock input to the PLL is INCLK0, with clock period 4000 ps.

Fig. 1 : Wave showing NORMAL mode operation.

When input port ARESET is asserted, it will cause the LOCKED port and all CLK outputs to drop to zero. The PLL will relock to the input clock when this port is deasserted.
